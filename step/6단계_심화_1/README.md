# 6단계 심화 1 학습 메모

심화 1 단계를 풀면서 자주 쓰는 문법과 핵심 개념을 정리합니다.

## 1) `replace`와 정규식으로 여러 패턴을 한 번에 바꾸기

- 문자열에서 **여러 종류의 부분 문자열**을 같은 규칙으로 처리해야 할 때, `replace`의 첫 인자에 **정규식**을 넘기면 한 번에 치환할 수 있습니다. (예: `2941번` 크로아티아 알파벳 — `c=`, `c-`, `dz=`, `d-`, `lj`, `nj`, `s=`, `z=` 각각을 한 글자로 세기 위해, 해당 부분을 임의의 한 문자로 바꾼 뒤 길이를 재는 방식)
- `/ ... /`는 **정규식 리터럴**로, 그 안의 패턴이 문자열에서 찾아 바꿀 부분이며, `|`(파이프)는 정규식에서 **또는(or)** 이므로, 아래처럼 쓰면 나열된 패턴 중 **어느 하나라도** 일치하면 치환 대상이 됩니다.

```js
const croatian = "ljes=njak";
const normalized = croatian.replace(/c=|c-|dz=|d-|lj|nj|s=|z=/g, "*");
console.log(normalized); // "*e**ak"
```

## 2) 전역 플래그 `/g`: 문자열 **전체**에서 **모두** 치환

- 정규식 끝의 `g`는 **global(전역)** 을 뜻합니다.
- `g`가 있으면 `replace`는 문자열 **처음부터 끝까지** 스캔하면서, 일치하는 부분을 **모두** 지정한 문자열(또는 함수 반환값)으로 바꿉니다. “문자열 안에 같은 패턴이 여러 번 나와도 전부 바꾸고 싶다”면 반드시 **`/패턴/g`** 형태로 **`g`를 붙이는지** 확인하는 것이 좋습니다.
- `g`가 없으면 `replace`는 **첫 번째 일치만** 바꾸고 멈춥니다. 나머지 같은 패턴은 그대로 남습니다.

```js
const s = "aa";
console.log(s.replace(/a/, "b")); // "ba" — 첫 a만 b로
console.log(s.replace(/a/g, "b")); // "bb" — 모든 a를 b로
```
